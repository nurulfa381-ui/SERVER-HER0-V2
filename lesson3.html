📘 What is Active Directory?

Active Directory Domain Services (AD DS) is a Microsoft Server Role used to manage users, computers, groups, printers and security policies within a Windows Domain Network.

⚙ Installation Steps

  1. Open Server Manager
  2. Select Add Roles and Features
  3. Select Active Directory Domain Services
  4. Click Install
  5. Restart Server if required
