📘 What is DNS?

Domain Name System (DNS) is a network service that translates domain names into IP addresses. DNS allows users and computers to locate servers quickly without remembering numeric IP addresses.

✅ Why DNS is Important?

  • Converts domain names into IP addresses.
  • Allows clients to find the Domain Controller.
  • Supports Active Directory services.
  • Improves network communication.
  • Essential for Windows Domain environments.

⚙ Configuration Steps

  1. Open Server Manager.
  2. Select Tools → DNS.
  3. Expand the Server Name.
  4. Open Forward Lookup Zones.
  5. Create or verify the DNS Zone.
  6. Ensure Host (A) Records are available.
  7. Save the configuration.

💻 Verification Command

Open Command Prompt and run:

nslookup serverhero.local

If the DNS configuration is correct, the server IP address will be displayed.

💡 Mission Tips

  • DNS is automatically installed with AD DS in most cases.
  • Always verify the Forward Lookup Zone.
  • Check that the DNS service is running.
  • Use nslookup to test DNS resolution.
